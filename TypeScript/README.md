# TypeScript

This project covers TypeScript fundamentals including interfaces, classes, generics, namespaces, and DOM manipulation.

## Tasks

### task_0 - Creating an interface for a student

Defines a `Student` interface with `firstName`, `lastName`, `age`, and `location` fields. Creates two student instances, stores them in an array, and renders them in an HTML table using vanilla JavaScript/TypeScript.

## Requirements

- TypeScript compiler must show no warnings or errors
- All variables use TypeScript types where possible
- Built with Webpack + ts-loader
